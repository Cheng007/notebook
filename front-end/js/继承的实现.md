## 继承的实现

### 基于原型链的继承
```js
function f() {
  this.a = 1
  this.b = 2
}
const o = new f() // { a: 1, b: 2 }

// 不要在 f 函数的原型上直接定义 f.prototype = {}，这样会直接打破原型链
f.prototype.b = 3
f.prototype.c = 4
f.prototype.func = () => {
  // do something
}

// 整个原型链如下：
// o ---> f.prototype ----> Object.prototype ----> null

console.log(o.a) // 1
// a 是 o 的自身属性吗？是的，改属性的值为 1

console.log(o.b) // 2
// b 是 o 的自身属性吗？是的，该属性的值为 2
// 原型上也有一个'b'属性，但是它不会被访问到（属性遮蔽）

cosole.log(o.c) // 4
// c 是 o 的自身属性吗？不是，那看看它的原型上有没，找到了，为 4

console.log(o.d) // undefined
// d 在 o 的自身和原型链上直到最底层 null 都没找到，返回 undefined

```

### `calss`
```js
class Product {
  construct(name, price) {
    this.name = name
    this.price = price
  }
}

class Food extends Product {
  category = 'food'
}
class Toy extends Product {
  category = 'toy'
}

const cheese = new Food('feta', 5)
const fun = new Toy('robot', 40)
```

### 使用`call`方法调用父构造函数

```js
function Product(name, price) {
  this.name = name
  this.price = price
}

function Food(name, price) {
  Product.call(this, name, price)
  this.category = 'food'
}

function Toy(name, price) {
  Product.call(this, name, price)
  this.category = 'toy'
}

const cheese = new Food('feta', 5)
const fun = new Toy('robot', 40)
```
